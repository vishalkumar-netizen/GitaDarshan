Shrimad Bhagavad Gita — Sacred Scripture

A beautiful, single-file web application to read and explore the Shrimad Bhagavad Gita. It features all 18 chapters and 700 verses with multiple translations, transliterations, and word meanings. Includes daily wisdom, search, bookmarks, and deep customization (themes, fonts, and dark mode).

Features

Read & Explore: Access all 18 chapters and 700 verses seamlessly.

Multiple Translations: View verses translated by over 10 different revered scholars.

In-Depth Study: Includes Sanskrit shlokas, English transliteration, and word-by-word meanings.

Daily Wisdom: Discover a new, inspiring verse every day.

Search: Quickly find verses by searching for specific words, concepts, or phrases.

Bookmarks: Save your favorite verses to your personal collection (uses local storage).

Personalization Engine: - Toggle between Light and Dark modes.

Choose from 10 distinct color themes (Bhagwa, Forest, Navy, Sepia, etc.).

Select from 6 beautifully legible font families.

Adjust reading size for the perfect viewing experience.

How to Run

Because this is a completely self-contained, single-file web application, running it is incredibly simple:

Download or clone this repository.

Open Gita3.html directly in any modern web browser (Chrome, Firefox, Safari, Edge).

No build tools, server, or dependencies are required!

Data Source & Credits

The verse data, translations, and word meanings are dynamically fetched from the open-source VedicScriptures API.

Font assets are served via Google Fonts.
